# GameRPG

Консольна рольова гра на C++ з покроковими боями, процедурно генерованими лабіринтами та фінальним босом.

---

## Ігровий процес

Гравець проходить **3 рівні** — кожен являє собою лабіринт 5×5 з різними кімнатами. Після кожного рівня відкривається **крамниця**, де можна витратити зібране золото. Після третього рівня чекає **фінальний бій з босом**.

### Типи кімнат

| Символ | Тип | Опис |
|--------|-----|------|
| `@` | Гравець | Поточна позиція |
| `E` | Ворог | Бій із противником |
| `T` | Пастка | Завдає шкоди при вході |
| `C` | Скриня | Предмет або золото |
| `F` | Джерело | Відновлює HP |
| `S` | Святиня | Бонус до характеристик |
| `X` | Вихід | Перехід на наступний рівень |
| `·` | Порожня | Нічого не відбувається |

### Вороги

- **Слабкий** — мало HP, не використовує спецатаки
- **Звичайний** — збалансований противник
- **Еліта** ★ — підвищені характеристики, спецатаки
- **Бос** ⚠ — унікальний противник у кімнаті

### Система бою

У звичайному бою доступні чотири дії:

1. **Атакувати** — звичайна атака (ATK ± 2)
2. **Навичка** — подвійний удар (x2 ATK), перезарядка 3 ходи
3. **Захист** — наступна атака ворога завдає -50% шкоди
4. **Втекти** — 40% шанс успіху (проти босів — 10%)

### Фінальний бос — Зарахель, Архідемон

Бій проходить у **3 фазах**:

| Фаза | Умова | Особливості |
|------|-------|-------------|
| 1 — Нормальна | HP > 50% | Щит темряви, заряд атаки |
| 2 — Лють ⚡ | HP 20–50% | +12 ATK, вкрадення HP, подвійний удар |
| 3 — Відчай ☠ | HP < 20% | +10 ATK, апокаліптичні атаки, самозцілення |

Проти боса доступні 5 дій: атака, подвійний удар, **ультимативний удар** (×3 ATK, ігнорує броню, кд 5), захист (-70% шкоди), використання зілля.

## Предмети

| Тип | Ефект |
|-----|-------|
| Зілля (CONSUMABLE) | Відновлення HP |
| Зброя (WEAPON) | Постійний бонус до ATK |
| Броня (ARMOR) | Постійний бонус до DEF |
| Аксесуар (ACCESSORY) | Особливий ефект |

---

## Управління

**На карті:**

| Клавіша | Дія |
|---------|-----|
| `W` `A` `S` `D` | Рух по лабіринту |
| `I` | Відкрити інвентар |
| `Q` | Пауза / вихід до меню |

**У бою:** вводити номер дії з меню (`1`–`4`, у бою з босом `1`–`5`).

## Збірка

**Вимоги:**
- C++20
- CMake ≥ 4.1
- Компілятор з підтримкою C++20 (GCC 10+, Clang 12+, MSVC 2019+)

**Кроки:**

```bash
git clone https://github.com/your-username/GameRPG.git
cd GameRPG
cmake -B build
cmake --build build
./build/GameRPG
```

Або відкрийте проєкт у **CLion** — він підхопить `CMakeLists.txt` автоматично.

## Технічні деталі

- Мова: **C++20**
- Збірка: **CMake + Ninja**
- Інтерфейс: консоль (ASCII-графіка, HP-бари з `█░`)
- Генерація рівнів: псевдовипадкова (`rand()` із seed від часу)
- Пам'ять: `Room` використовує move-семантику; `Enemy*` — сирий pointer із RAII-деструктором
